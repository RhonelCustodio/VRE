# Victoria Rondalla Ensemble

A modern web application showcasing the Victoria Rondalla Ensemble from Victoria, Tarlac, Philippines. This application preserves and promotes Filipino musical heritage through traditional instruments and timeless folk songs.

## Features

- **Instruments Gallery**: Browse traditional Rondalla instruments including Bandurria, Laud, Octavina, Guitar, Ukulele, and Double Bass
- **Folk Songs Library**: Explore Filipino folk songs including Lupang Hinirang, Bahay Kubo, Leron Leron Sinta, and more
- **Chord Charts**: Access chord progressions for learning traditional songs
- **Contact Form**: Functional contact form integrated with database for inquiries and bookings
- **Responsive Design**: Beautiful design that works on all devices

## Technology Stack

- **Frontend**: Vite, TypeScript, Tailwind CSS
- **Database**: Supabase (PostgreSQL)
- **Hosting**: Static site deployment ready

## Database Schema

The application uses four main tables:
- `instruments` - Information about traditional Rondalla instruments
- `folk_songs` - Details about Filipino folk songs
- `chord_charts` - Chord progressions for different instruments
- `contact_messages` - Contact form submissions

## Getting Started

The database is already configured with sample data including 6 instruments, 7 folk songs, and 3 chord charts.

The application will automatically load and display content from the database when you run the development server.
